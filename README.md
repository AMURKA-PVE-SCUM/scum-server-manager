## Возможности

- **Управление сервером**: запуск, остановка, перезапуск, мониторинг RAM/аптайма/игроков
- **SteamCMD**: установка и обновление сервера (app 3792580) с открытой консолью
- **Авто-рестарт при краше**: сервер перезапускается сам, если упал
- **Веб-панель** на порту 8080: управление сервером из браузера
- **Конфигурация**: визуальный редактор ServerSettings.ini, GameUserSettings.ini, EconomyOverride.json, RaidTimes.json
- **Расписание**: перезапуски по времени, роботы (Sentry) по дням
- **База данных SCUM.db**: игроки, кошелёк, навыки, атрибуты, отряды, транспорт
- **Администраторы**: добавление/удаление админов с флагами (SetGodMode, RestartServer)
- **Бэкапы** с авто-очисткой, восстановлением и расписанием
- **Discord-уведомления**: входы игроков, логи админа, чат, статус сервера
- **FTP-сервер**: встроенный доступ к файлам сервера
- **Мониторинг логов** в реальном времени с фильтрацией по типу
- **Файловый менеджер**: просмотр и редактирование конфигов
- **Перевод**: русский / английский

## Системные требования

| Компонент | Требование |
|-----------|-----------|
| ОС | Windows 10 / Windows Server 2019+ |
| RAM | 512 МБ (без учёта сервера SCUM) |
| Диск | 200 МБ (приложение) + ~20 ГБ (сервер) |

## Установка

Скачайте последний релиз со страницы [Releases](https://github.com/tolyan28rus/scum-server-manager/releases), распакуйте и запустите `SCUM Server Manager.exe`.

## Разработка

```bash
npm install
npm run build
npx electron dist/main/index.js
```

## Сборка релиза

```bash
npm run dist:win
```

Готовый `.exe` появится в папке `release/`.
