## Установка

### 1. RCON мод (обязательно)

Для работы RCON и чат-команд требуется мод от **herbie96x**:

**Репозиторий**: [github.com/herbie96x/SCUM-RCON](https://github.com/herbie96x/SCUM-RCON)

Установка:
1. Скачайте **UE4SS** (Unreal Engine 4 Scripting System)
2. Скачайте **scum_rcon** из репозитория выше
3. Установите в папку сервера:
   ```
   SCUM/SCUM/Binaries/Win64/
     ├── ue4ss.dll
     └── Mods/scum_rcon/...
   ```
4. Настройте `Mods/scum_rcon/config.ini`:
   ```ini
   [RCON]
   RCON_Port=28015
   RCON_Password=your_password
   RCON_Enabled=true
   ```
5. Перезапустите сервер

### 2. Менеджер

1. Скачайте последний [релиз](https://github.com/tolyan28rus/scum-server-manager/releases)
2. Распакуйте архив, запустите `SCUM Server Manager.exe`
3. На вкладке **Server Install** установите SteamCMD и сервер
4. Во вкладке **Web Panel** включите веб-панель (порт 8080)
5. Подключите RCON (host: `localhost`, порт: `28015`, пароль из config.ini)

Всё, менеджер готов к работе.

## Возможности

### Управление сервером
- Запуск / остановка / перезапуск сервера
- Мониторинг в реальном времени: RAM, аптайм, FPS, количество игроков
- Авто-старт при запуске приложения
- Авто-перезапуск при краше (с защитой от циклических рестартов)
- Параметры запуска: порт, QueryPort, MaxPlayers, -NoBattlEye, -fileopenlog
- Скрытие консоли сервера после запуска

### SteamCMD
- Автоматическая установка SteamCMD
- Проверка обновлений сервера (сравнение Build ID)
- Обновление сервера с прогресс-баром в реальном времени
- Блокировка авто-обновления SteamCMD
- Ручная переустановка сервера

### RCON
- Подключение / отключение по TCP (Source RCON протокол)
- Авто-переподключение при обрыве
- Отправка любых RCON команд
- Логирование всех команд в `logs/rcon_commands.log`

### Веб-панель (порт 8080)
- **Дашборд**: статус сервера, управление, live-консоль (SSE)
- **RCON консоль**: ввод команд, история (стрелки), быстрые команды
- **Игроки**: онлайн-список с автообновлением, карточки игроков
- **Наборы**: стартовый и ежедневный наборы, телепорт, VIP, SaveHome
- **WARGM**: настройка API, карточки товаров, тест подключения
- **Настройки**: путь SteamCMD, путь сервера
- Авторизация по Bearer токену (логин/пароль)
- Если RCON уже подключён в приложении — вход без авторизации

### Карточка игрока (веб-панель)
- Просмотр атрибутов, навыков, инвентаря, баланса, золота, Fame
- **Действия**: GodMode, Immortality, NamePlates, Kill, Silence, Knockout
- **Выдача**: предметы (автокомплит из iditem.txt), ТС, деньги, золото, Fame
- **Навыки**: установка всех 22 навыков на макс (одной кнопкой)
- **Атрибуты**: сила/ловкость/выносливость/интеллект (1-10)
- **Сообщения**: Announce (глобал), Notify (Info/Warning/Error/Success), Chat
- **Телепорт**: по координатам или в сохранённые локации
- **Сброс карго**: авто-определение координат игрока через ListPlayers

### Десктопное приложение (19 страниц)
- **Dashboard** — управление сервером, статус, launch args
- **Server Install** — установка SteamCMD и сервера
- **Scheduler** — авто-рестарты (по времени или интервалу), роботы Sentry
- **Web Panel** — вкл/выкл, порт, логин/пароль
- **Server Settings** — путь сервера, SteamCMD, порты
- **Game Settings** — редактор ServerSettings.ini
- **Economy Settings** — редактор EconomyOverride.json
- **Raid Settings** — редактор RaidSettings.json
- **Players** — поиск игроков в БД, просмотр/редактирование
- **Online Players** — онлайн-список, действия (атрибуты, GodMode, Kill, Silence, Knockout, Announce, Notify, Chat)
- **Squads** — просмотр отрядов из БД
- **Vehicles** — просмотр транспорта из БД
- **Loot Editor** — редактор .ini/.json файлов конфигов
- **Log Monitor** — просмотр логов по типам (admin, chat, login, vehicle, system)
- **Backup Manager** — создание/восстановление/удаление бэкапов, авто-очистка
- **Discord** — 5 вебхуков (admin, chat, vehicle, login, server status)
- **FTP** — встроенный FTP-сервер для доступа к файлам сервера
- **File Manager** — просмотр и редактирование файлов сервера
- **Settings** — язык (RU/EN), тема

### База данных SCUM.db (чтение и запись)
- **Игроки**: профили, кошелёк (normal + gold), Fame, флаги
- **Навыки**: все 22 навыка с уровнем и опытом
- **Атрибуты**: сила, телосложение, ловкость, интеллект, вес, рост, возраст
- **Инвентарь**: рекурсивное дерево предметов, быстрые слоты
- **Отряды**: список, участники, ранги, счёт
- **Транспорт**: все заспавненные ТС, состояние
- **Флаги**: базовые флаги с владельцами
- **Банк-аккаунты**: все счета с балансами
- **Таблица экономики**: топ игроков по деньгам/Fame
- **Редактирование**: баланс, Fame, навыки, атрибуты

### Чат-команды (внутриигровые)

| Команда | Алиасы | Описание |
|---------|--------|----------|
| `!help` | `!помощь`, `!commands` | Список команд |
| `!balance` | `!баланс` | Баланс (деньги + золото) |
| `!location` | `!локация`, `!координаты` | Координаты игрока |
| `!online` | `!онлайн` | Список игроков онлайн |
| `!startpack` | `!стартпак` | Стартовый набор (с кулдауном) |
| `!dailypack` | `!дейлипак` | Ежедневный набор (24h кулдаун) |
| `!wargm` | `!варгм` | Проверка/получение покупок из WARGM |
| `!teleport` | `!телепорт` | Список доступных телепортов |
| `!vip` | `!вип` | Статус VIP + оставшиеся дни |
| `!nameplates` | `!ники` | Включить никнеймы над головами |
| `!savedom` | `!сохранитьдом` | Сохранить текущую позицию как дом |
| `!home` | `!дом` | Телепорт домой |
| `!homes` | `!дома` | Список сохранённых домов |

### Наборы (Loot Packs)
- **Стартовый набор**: предметы + количество, кулдаун (0 = один раз, >0 = через N часов)
- **Ежедневный набор**: предметы + количество, 24h кулдаун
- **VIP-бонусы**: дополнительные предметы/деньги при получении наборов
- Персистентные кулдауны (сохраняются в `logs/pack_cooldowns.json`)
- Настройка через веб-панель с автокомплитом предметов

### WARGM — магазин + автовыдача
Интеграция с [WARGM.RU](https://wargm.ru):
- **Карточки товаров** — привязка `offer_id` из магазина к правилам выдачи
- **Типы выдачи**: предметы, транспорт, навыки (все или один), атрибуты, деньги, золото, карго-дроп, VIP
- **Автоматическая выдача** — игрок пишет `!wargm` → получает все купленные товары
- **Защита от дубликатов** — `operation_claim` + таблица `wargm_deliveries`
- **Rate-limit**: настраиваемый кулдаун между проверками
- **Экспорт/импорт** карточек в JSON
- **Логирование** API-запросов и выдач (ежедневные файлы)
- **Дебаг**: просмотр сырых операций, проверка игрока

### VIP система
- Добавление VIP игроков с указанием SteamID и срока действия
- Бонусы при получении стартового и ежедневного наборов (деньги, золото, Fame, предметы)
- Проверка статуса через `!vip`

### Телепорты
- Настраиваемые точки телепорта (название, X/Y/Z, цена)
- Телепорт с задержкой 15 секунд
- Стоимость списывается с баланса игрока

### SaveHome (сохранение дома)
- Игроки могут сохранять текущую позицию (`!savedom`)
- Телепорт домой (`!home`) с настраиваемой ценой
- Ограничение по количеству домов (VIP — больше)
- Данные сохраняются в `data/home_locations.json`

### Расписание (Scheduler)
- **Авто-рестарты**: по расписанию (точное время) или интервалу
- **Роботы Sentry**: включение/отключение по дням недели и времени
- Гибкая настройка дней недели (чипсы)

### Discord-уведомления
5 типов вебхуков:
- Admin Log — действия админов
- Chat Messages — сообщения в чате
- Vehicle Events — события с ТС
- Login/Logout — вход/выход игроков
- Server Status — запуск/остановка/перезапуск сервера

### FTP-сервер
- Встроенный FTP-сервер для доступа к файлам сервера
- Настраиваемый порт, логин/пароль
- Chroot в корневую папку сервера
- Passive mode с настраиваемым PASV URL

### Бэкапы
- Автоматические по расписанию (настраиваемый интервал)
- Авто-очистка старых бэкапов
- Создание/восстановление/удаление одной кнопкой
- Ручные именованные бэкапы

### Управление администраторами
- Добавление/удаление в `AdminUsers.ini`
- Настройка флагов: SetGodMode, SetTime, RestartServer, Kick, Ban и др.
- Синхронизация с файлом конфигурации

### Конфигурация
Визуальные и текстовые редакторы:
- `ServerSettings.ini` — основные настройки
- `GameUserSettings.ini` — игровые настройки
- `EconomyOverride.json` — экономика
- `RaidSettings.json` — рейд-таймы

### Локализация
Русский / Английский — переключается в боковом меню.

## Разработка и сборка

```bash
git clone https://github.com/tolyan28rus/scum-server-manager.git
cd scum-server-manager
npm install
npm run build
npx electron dist/main/index.js
```

### Сборка portable

```bash
npm run dist:win
```

Готовый .exe появится в папке `release/`.
