## Установка

### 1. RCON мод (обязательно)

Для работы RCON и чат-команд требуется мод от **herbie96x**:

**Репозиторий**: [github.com/herbie96x/SCUM-RCON](https://github.com/herbie96x/SCUM-RCON)

Установка:
1. Скачайте **UE4SS** (Unreal Engine 4 Scripting System) — [github.com/UE4SS-RE/RE-UE4SS](https://github.com/UE4SS-RE/RE-UE4SS)
2. Скачайте **scum_rcon** из репозитория выше
3. Установите в папку сервера:
   ```
   SCUM/SCUM/Binaries/Win64/
     ├── ue4ss.dll
     └── Mods/scum_rcon/...
   ```
4. Настройте `Mods/scum_rcon/config.ini`:
   ```ini
   [RCON]
   RCON_Port=28015
   RCON_Password=your_password
   RCON_Enabled=true
   ```
5. Перезапустите сервер

### 2. Менеджер

1. Скачайте последний [релиз](https://github.com/tolyan28rus/scum-server-manager/releases)
2. Установите `SCUM Server Manager Setup X.X.X.exe`
3. На вкладке **Server Install** установите SteamCMD и сервер
4. Во вкладке **Web Panel** включите веб-панель (порт 8080)
5. Подключите RCON (host: `localhost`, порт: `28015`, пароль из config.ini)

Всё, менеджер готов к работе.

## Возможности

### Управление сервером
- Запуск / остановка / перезапуск сервера
- Мониторинг в реальном времени: RAM, аптайм, FPS, количество игроков
- Авто-старт при запуске приложения, авто-перезапуск при краше (с защитой от циклических рестартов)
- Параметры запуска: порт, QueryPort, MaxPlayers, -NoBattlEye, -fileopenlog
- Скрытие консоли сервера после запуска

### SteamCMD
- Автоматическая установка SteamCMD
- Проверка обновлений сервера (сравнение Build ID через api.steamcmd.net)
- Обновление сервера с прогресс-баром в реальном времени (3 попытки, авто-восстановление)
- Блокировка авто-обновления SteamCMD
- Ручная переустановка сервера

### RCON (Source RCON протокол)
- Подключение / отключение, авто-переподключение при обрыве
- Отправка любых RCON команд через веб-панель или десктопный интерфейс
- Логирование всех команд в `logs/rcon_commands.log`

### Веб-панель (HTTP, порт 8080)
Полноценный веб-интерфейс с авторизацией по Bearer-токену:

#### Дашборд
- Статус сервера, онлайн, аптайм, RAM, RCON
- Live-консоль (SSE), вывод лога сервера в реальном времени
- **Версия приложения** + кнопка **Проверить обновления** (с автообновлением через GitHub Releases)

#### Карта
- 3 варианта карты: цветная, серая, оранжевая (из `SCUM-Images/map/`, 4096x4096)
- **Игроки**: позиции в реальном времени (через `ListPlayers`)
- **Флаги территорий**: из `SCUM.db`
- **Транспорт**: все ТС на карте
- **Секторная сетка**: D4–Z0 как в игре, сектора 5×5
- Контекстное меню ПКМ: телепорт по координатам, копирование координат

#### Игроки
- Онлайн-список с автообновлением (5 сек)
- Карточки игроков: атрибуты, навыки, инвентарь, баланс, золото, Fame
- **Действия**: GodMode, Immortality, NamePlates, Kill, Silence, Knockout, Kick, Ban
- **Выдача**: предметы (автокомплит из `iditem.txt`), ТС, деньги, золото, Fame
- **Навыки**: установка всех 22 навыков на макс (одной кнопкой)
- **Атрибуты**: сила/ловкость/выносливость/интеллект (1–10)
- **Сообщения**: Announce (глобал), Notify (Info/Warning/Error/Success), Chat (в раздельный канал)
- **Телепорт**: по координатам, в сохранённые локации, к игроку
- **Сброс карго**: авто-определение координат игрока через `ListPlayers`

#### Чат
- SSE-трансляция игрового чата в реальном времени
- Фильтры по каналам (Global, Admin, Local)
- Отправка сообщений от имени бота с выбором ника отправителя
- Кодировка UTF-16LE с авто-выравниванием

#### Модули
- **Наборы (Packs)**: стартовый и ежедневный наборы с кулдаунами, VIP-бонусы
- **VIP**: добавление игроков со сроком, бонусы к наборам
- **Телепорты**: точки с названием, координатами и ценой
- **SaveHome**: сохранение позиции, телепорт домой
- **Аирдроп**: ручной и авто-дроп (`SpawnInventoryFullOf`), система калибровки высоты (125 точек), авто-дроп по таймеру и мин. игрокам
- **Рейтинг**: учёт времени онлайн, денег, золота, Fame, таблица лидеров, чёрный список, `!рейтинг` в чате
- **Награды**: почасовые (за час онлайн) и топ-игроков (за период с множителем), `#AddGold`/`#AddMoney`/`#AddFame`
- **Chat Sender**: настройка ника отправителя в чате

#### WARGM — магазин + автовыдача
Интеграция с [WARGM.RU](https://wargm.ru):
- **Карточки товаров** — привязка `offer_id` к правилам выдачи
- **Типы выдачи**: предметы, транспорт, навыки (все/один), атрибуты, деньги, золото, **слава (Fame)**, карго-дроп, VIP
- Автоматическая выдача по `!wargm` / `!варгм`
- Защита от дубликатов (operation_claim + БД wargm_deliveries)
- Rate-limit, экспорт/импорт карточек JSON, дебаг-режим
- Логирование API-запросов и выдач (ежедневные файлы `logs/`)

#### Настройки
- Путь SteamCMD, путь сервера, авто-детект
- Web Panel: вкл/выкл, порт, логин/пароль
- Локализация RU/EN

### Десктопное приложение (19 страниц)
- **Dashboard** — управление сервером, статус, launch args
- **Server Install** — установка SteamCMD и сервера с прогрессом
- **Scheduler** — авто-рестарты (по точному времени / интервалу), роботы Sentry по дням недели
- **Web Panel** — вкл/выкл, порт, логин/пароль
- **Server Settings** — пути, порты, MaxPlayers, -NoBattlEye
- **Game Settings** — визуальный редактор `ServerSettings.ini`
- **Economy Settings** — редактор `EconomyOverride.json`
- **Raid Settings** — редактор `RaidSettings.json`
- **Players** — поиск игроков в `SCUM.db`, просмотр/редактирование профиля
- **Online Players** — онлайн с действиями (атрибуты, GodMode, Kill, Silence, Announce, Chat)
- **Squads** — просмотр отрядов из БД
- **Vehicles** — просмотр транспорта из БД
- **Loot Editor** — редактор .ini/.json файлов сервера
- **Log Monitor** — просмотр логов по типам (admin, chat, login, vehicle, system)
- **Backup Manager** — ручные/авто-бэкапы с ротацией, восстановление
- **Discord** — 5 вебхуков (admin, chat, vehicle, login, server status)
- **FTP** — встроенный FTP-сервер для доступа к файлам сервера
- **File Manager** — просмотр и редактирование файлов
- **Settings** — язык (RU/EN), тема

### База данных SCUM.db (чтение + запись)
- Игроки: профили, кошелёк (normal + gold), Fame, флаги
- Навыки: все 22 навыка с уровнем и опытом
- Атрибуты: сила, телосложение, ловкость, интеллект, вес, рост, возраст
- Инвентарь: рекурсивное дерево предметов, быстрые слоты
- Отряды: список, участники, ранги, счёт
- Транспорт: все ТС, состояние
- Флаги территорий с владельцами
- Банк-аккаунты с балансами
- Топ экономики по деньгам/Fame
- Редактирование: баланс, Fame, навыки, атрибуты

## Разработка и сборка

```bash
git clone https://github.com/tolyan28rus/scum-server-manager.git
cd scum-server-manager
npm install
npm run build
npx electron dist/main/index.js
```

### Сборка установщика

```bash
build.bat
```

Готовый .exe появится в папке `release/`.

### Замечания по сборке

- **winCodeSign**: на некоторых версиях Windows (10.0.26200+) 7-Zip не может распаковать macOS symlink из архива winCodeSign. В `package.json` установлен `"signAndEditExecutable": false` — подпись отключена, это не влияет на работу.
- **SCUM-Images**: не включены в git-репозиторий (2470 файлов, ~204 МБ). Копируются в установщик через `extraResources`.
