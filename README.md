# 🎯 SCUM Server Manager

**Управление SCUM Dedicated Server под Windows — десктопное приложение + веб-панель**

![Version](https://img.shields.io/badge/version-1.3.0-green)
![License](https://img.shields.io/badge/license-MIT-green)

SCUM Server Manager — это полнофункциональный инструмент для установки, настройки и управления выделенным сервером SCUM. Работает как десктопное приложение (Electron) и как веб-панель, доступная из браузера.

## 🚀 Возможности

### Управление сервером
- **Запуск / Остановка / Перезапуск** сервера одной кнопкой
- **Мониторинг**: RAM, аптайм, количество игроков в реальном времени
- **Авто-рестарт при краше**: сервер перезапускается автоматически, если упал
- **Статус-бар** с иконкой в трее

### SteamCMD
- **Установка** сервера SCUM (app 3792580) через встроенный SteamCMD
- **Обновление** с открытой консолью — видно весь процесс
- **Прогресс** установки отображается на дашборде

### Веб-панель
- Полноценная веб-панель на порту 8080
- Управление сервером из браузера (вкл/выкл, перезапуск)
- **Консоль** с лайв-выводом логов сервера
- **Авторизация** по токену

### Конфигурация
Визуальные редакторы для всех конфигов сервера:
- **ServerSettings.ini** — основные настройки
- **GameUserSettings.ini** — игровые настройки
- **EconomyOverride.json** — экономика (стоимость, спаун, крафт)
- **RaidSettings.json** — рейд-таймы (окна рейдов по дням недели)

### База данных SCUM.db
Просмотр и редактирование встроенной БД:
- **Игроки**: управление кошельком, флаг FreeForAll
- **Навыки**: просмотр и редактирование
- **Атрибуты**: сила, выносливость и т.д.
- **Отряды**: список, участники
- **Транспорт**: информация о транспорте

### Администраторы
- Добавление / удаление администраторов
- **Флаги**: `SetGodMode`, `RestartServer`
- Автоматическая синхронизация с `AdminUsers.ini`
- Удобный интерфейс с выбором флагов

### Расписание
- **Перезапуски сервера** по расписанию
- **Роботы (Sentry)** по дням недели + время
- **Статистика**: когда был последний запуск/остановка

### Бэкапы
- Автоматические бэкапы по расписанию
- **Авто-очистка** старых бэкапов
- Восстановление из любого бэкапа одной кнопкой
- Прогресс создания/восстановления

### Discord-уведомления
Настраиваемые уведомления через Webhook:
- **Вход игрока** на сервер
- **Выход игрока** с сервера
- **Админ-логи** (изменения в игре)
- **Чат** (сообщения игроков)
- **Статус сервера** (вкл/выкл/аптайм)
- **Изменение конфигов**
- **Авто-рестарт**

### FTP-сервер
- Встроенный FTP-сервер для доступа к файлам
- Настройка порта, папки, логина/пароля
- Удобно для загрузки модов и конфигов

### Логи
- Мониторинг логов в реальном времени
- Фильтрация по типу (чат, админ, подключения, ошибки)
- Авто-обновление

### Файловый менеджер
- Просмотр файловой системы сервера
- Редактирование конфигов онлайн
- Загрузка и скачивание файлов

### Локализация
- Русский / Английский — переключается в интерфейсе

## 📦 Установка

1. Скачайте последний релиз со страницы [Releases](https://github.com/tolyan28rus/scum-server-manager/releases)
2. Распакуйте архив (рекомендуется [7-Zip](https://www.7-zip.org/) — разархивирует быстрее)
3. Запустите `SCUM Server Manager.exe`

При первом запуске потребуется:
1. Указать путь до SteamCMD (или скачать автоматически)
2. Установить SCUM Dedicated Server через вкладку **Server Install**
3. Настроить конфиги сервера

---

## 🏁 Быстрый старт

1. Откройте вкладку **Server Install**
2. Нажмите **Install SteamCMD**, затем **Install Server**
3. Дождитесь установки (откроется консоль SteamCMD)
4. Перейдите в **Server Settings** и укажите параметры сервера
5. Нажмите **Save & Start** на дашборде

Сервер запущен! Теперь можно:
- Настроить **Game Settings** (игровые настройки)
- Настроить **Economy** (экономику)
- Настроить **Raid Settings** (рейд-таймы)
- Добавить **админов** через вкладку Players
- Настроить **авто-рестарт** в настройках

---

## 🌐 Веб-панель

Веб-панель доступна по адресу: `http://<ваш-ip>:8080`

Настройка:
1. Откройте вкладку **Web Panel**
2. Укажите порт (по умолчанию 8080)
3. Установите токен для авторизации
4. Нажмите **Start**

После запуска веб-панели можно управлять сервером из любого браузера в локальной сети.

## ⚙️ Системные требования

| Компонент | Требование |
|-----------|-----------|
| **ОС** | Windows 10 / Windows Server 2019+ |
| **RAM** | 512 МБ (без учёта сервера SCUM) |
| **Диск** | 200 МБ (приложение) + ~20 ГБ (сервер) |
| **Интернет** | Для установки/обновления сервера через SteamCMD |

## 🛠️ Разработка

```bash
# Клонировать репозиторий
git clone https://github.com/tolyan28rus/scum-server-manager.git
cd scum-server-manager

# Установить зависимости
npm install

# Собрать проект
npm run build

# Запустить
npx electron dist/main/index.js
```

## 📦 Сборка релиза

```bash
# Собрать portable версию
npm run dist:win
```

Готовый архив появится в папке `release/`.
